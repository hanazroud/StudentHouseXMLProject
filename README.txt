Mon rojet soa : StudentHouse – Gestion des logements étudiants
Nom et prenom :  Zroud Hana

Description :
Mon projet XML modélise la gestion des résidences universitaires et des étudiants logés.
Il comprend les résidences,les chambres,les étudiants et leurs paiements.

Les fichiers correspond :
- student_house.xml : données principales
- student_house.dtd : validation DTD
- student_house.xsd : validation XSD
- student_house.xsl : transformation HTML
- README.txt : documentation du projet

Utilisation :
1. Valider le fichier XML avec le DTD et le XSD.
2. Appliquer le fichier XSL pour transformer les données en page HTML.
3. Ouvrir le résultat dans un navigateur web pour visualiser la gestion complète.
